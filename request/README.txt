

Planetary and House Packages Readme
------------------------
OS Compatible - Linux OS Server
Programming Language - PHP (Version 5.4+)

Steps to use the package
------------------------------
1. Place the "package" folder in your server directory. We do not recommend to put this folder in public (public_html/www) folder.
2. You might not need to change anything in "package". Please use the provided package as is.
3. test.php is given as an example of how can the package folder be utilised for getting planetary positions and house cusps details.
4. Please shoot an email at mail@astrologyapi.com for any query regarding the package.
